# Auth Components

Componentes de autenticação do aplicativo.

## LoginScreen

Componente de tela de login com design moderno e suporte a tema claro/escuro.

### Uso

```tsx
import { LoginScreen } from '@/auth';

export default function Login() {
  return <LoginScreen />;
}
```

### Características

- Design responsivo
- Suporte a tema claro/escuro
- Campos de email e senha
- Toggle de visibilidade de senha
- Botão de login com ícone
- Opção de login biométrico
- Link para recuperação de senha
- Link para criação de conta
